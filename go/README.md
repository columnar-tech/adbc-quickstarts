# Go ADBC Quickstarts

Simple Go examples showing how to use ADBC to connect, run a query, and return the results.

## Prerequisites

1. [Install Go](https://go.dev/doc/install)

1. [Install dbc](https://docs.columnar.tech/dbc/getting_started/installation/)

  - CockroachDB
